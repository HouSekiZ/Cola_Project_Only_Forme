# โปรเจคนี้เป็นส่วนหนึ่งของรายวิชา AI in Public Affairs CP413601

### ความเป็นมา
ประเทศไทยกำลังก้าวเข้าสู่สังคมสูงวัยระดับสุดยอด ส่งผลให้จำนวนผู้ป่วยติดเตียงที่มีภาวะพึ่งพิงสูงเพิ่มขึ้นอย่างต่อเนื่อง ซึ่งผู้ป่วยกลุ่มนี้เผชิญความเสี่ยงจากการเกิดแผลกดทับหากไม่ได้รับการพลิกตัวตามกำหนด และมักมีข้อจำกัดด้านการสื่อสารที่ไม่สามารถตะโกนขอความช่วยเหลือเองได้เมื่อเกิดเหตุฉุกเฉิน ขณะที่การดูแลในปัจจุบันผ่าน อสม. หรืออุปกรณ์สวมใส่ยังมีข้อจำกัดเรื่องเวลาและภาระค่าใช้จ่าย โครงการนี้จึงพัฒนา ระบบเฝ้าระวังด้วยปัญญาประดิษฐ์ (AI) ที่สามารถตรวจจับท่าทางการนอน การขยับดวงตา และการเคลื่อนไหวมือเพื่อใช้เป็นสัญญาณขอความช่วยเหลือแทนการพูด พร้อมระบบแจ้งเตือนไปยังผู้ดูแลและ รพ.สต. แบบ Real-time เพื่อลดความเสี่ยงภาวะแทรกซ้อน เพิ่มประสิทธิภาพการดูแลเชิงรุก และยกระดับมาตรฐานสาธารณสุขไทยสู่ Smart Healthcare อย่างยั่งยืน

### วัตถุประสงค์ของโครงการ (Objectives)
1.	เพื่อพัฒนาระบบตรวจจับเหตุการณ์ผิดปกติของผู้ป่วยติดเตียงแบบอัตโนมัติ
2.	เพื่อพัฒนาระบบการแจ้งเตือนแบบเรียลไทม์ไปยังผู้ดูแล อาสาสมัครสาธารณสุขประจำหมู่บ้าน และบุคลากรทางการแพทย์ เมื่อผู้ป่วยต้องการความช่วยเหลือหรือเกิดภาวะฉุกเฉิน
3.	เพื่อเพิ่มความปลอดภัยให้แก่ผู้ป่วยติดเตียง ลดความล่าช้าในการเข้าช่วยเหลือ และสนับสนุนการดูแลระยะยาวอย่างมีประสิทธิภาพ
4.	เพื่อเพิ่มช่องทางการสื่อสารฉุกเฉินให้กับผู้ป่วยที่สื่อสารทางเสียงไม่ได้ ให้ได้รับความช่วยเหลืออย่างทันท่วงที

### ขอบเขตของโครงการ (Scope)
-	ขอบเขตด้านข้อมูล
  1.	ข้อมูลท่าทางร่างกาย (Pose Estimation)  เช่น การยกมือขอความช่วยเหลือและการกระพริบตา 
  2.	ข้อมูลที่ตำแหน่งหรือที่อยู่อาศัยของผู้ป่วยติดเตียง

-	ขอบเขตด้านพื้นที่
  1. ระบบเน้นติดตั้งใน บ้านหรือที่พักของผู้ป่วยติดเตียงที่อาศัยอยู่ตามลำพัง

-	ฟังก์ชันที่จะพัฒนา
  1.	ระบบตรวจจับพฤติกรรมและเหตุการณ์ผิดปกติ ระบบสามารถตรวจจับและจำแนกท่าทางของผู้ป่วยติดเตียงแบบอัตโนมัติ ได้แก่ ท่านอนหงาย (Supine Position) ท่านอนตะแคงซ้าย (Left-side Position) ท่านอนตะแคงขวา (Right-side Position)และการเคลื่อนไหวรุนแรงหรือผิดปกติ (Abnormal Movement Detection)
  2.	ระบบแจ้งเตือนอัจฉริยะแบบเรียลไทม์ ระบบมีการแจ้งเตือนอัตโนมัติในกรณีต่าง ๆ เช่น แจ้งเตือนเมื่อถึงเวลารับประทานยา แจ้งเตือนเมื่อถึงเวลาทำกายภาพบำบัด แจ้งเตือนเมื่อผู้ป่วยอยู่ในท่าเดิมนานเกินกำหนดและแจ้งเตือนขอความช่วยเหลือ รวมถึงสามารถแจ้งเตือนไปยังผู้ดูแลได้
  3.	ระบบบันทึกและวิเคราะห์ข้อมูล ระบบจะบันทึกข้อมูลเหตุการณ์ทั้งหมดลงในฐานข้อมูลกลาง เช่น เหตุการณ์ผิดปกติที่เกิดขึ้ระยะเวลาการตอบสนองของผู้ดูแล


### ผู้มีส่วนได้ส่วนเสีย (Stakeholder Analysis)
- ผู้ป่วยติดเตียง
    -	ผู้ได้รับประโยชน์โดยตรงจากระบบ
    -	มีความปลอดภัยเพิ่มขึ้น ลดความเสี่ยงแผลกดทับและเหตุฉุกเฉิน
-	ผู้ดูแลในครอบครัว
    -	รับการแจ้งเตือนแบบเรียลไทม์
    -	ลดภาระและความเครียดในการเฝ้าระวังตลอด 24 ชั่วโมง
-	อาสาสมัครสาธารณสุขประจำหมู่บ้าน (อสม.)
    -	ใช้ข้อมูลจากระบบในการติดตามอาการผู้ป่วย
    -	เพิ่มประสิทธิภาพการเยี่ยมบ้านและการประสานงานรวมถึงการเข้าดูแลผู้ป่วยติดเตียง
-	โรงพยาบาลส่งเสริมสุขภาพตำบล (รพ.สต.)
    -	รับข้อมูลเพื่อประเมินอาการและวางแผนการดูแล
    -	ลดการเข้ารักษาในโรงพยาบาลโดยไม่จำเป็น
-	องค์การบริหารส่วนตำบล (อบต.) / เทศบาล
    -	สนับสนุนงบประมาณและการดำเนินงานในระดับท้องถิ่น
    -	ใช้ข้อมูลเชิงสถิติในการวางแผนดูแลผู้สูงอายุในพื้นที่
-	หน่วยงานสาธารณสุขระดับจังหวัด/ประเทศ (ถ้าต้องการเพิ่มความครบ)
    -	ใช้ข้อมูลเชิงภาพรวมเพื่อกำหนดนโยบายดูแลผู้สูงอายุ
    -	สนับสนุนการพัฒนา Smart Healthcare ในระดับพื้นที่


### ประเด็นทางนโยบาย (Policy Issues)
 - การส่งเสริมการใช้เทคโนโลยีสิ่งอำนวยความสะดวก (Assistive Technology): การผลักดันให้ AI เป็นส่วนหนึ่งของสวัสดิการการดูแลผู้ป่วยระยะยาว
 - มาตรฐานการดูแลผู้ป่วยติดเตียงในระดับชุมชน: การใช้เทคโนโลยีช่วยลดช่องว่างในกรณีที่ผู้ดูแลมีจำนวนน้อยกว่าผู้ป่วย
 - ความเป็นส่วนตัวและสิทธิผู้ป่วย (PDPA): การติดตั้งกล้องในระยะประชิดเพื่อตรวจจับใบหน้าและมือ ต้องมีความปลอดภัยของข้อมูลสูงสุดและทำเพื่อวัตถุประสงค์ช่วยชีวิตเท่านั้น

### การบริการสาธารณะ (Public Services)
- ระบบเรียกขานฉุกเฉินอัจฉริยะ (Smart Emergency Call): เปลี่ยนกล้องวงจรปิดทั่วไปให้เป็นปุ่มกดเรียกฉุกเฉินที่ทำงานด้วยการขยับร่างกาย
- การยกระดับคุณภาพชีวิต (Quality of Life): ลดความรู้สึกโดดเดี่ยวและความวิตกกังวลของผู้ป่วยติดเตียงที่ไม่สามารถสื่อสารได้
- การลดภาระภาครัฐ: ลดการสูญเสียทรัพยากรบุคคลและงบประมาณจากการช่วยเหลือที่ล่าช้าเกินไป

### จุดที่เน้นเป็นพิเศษ:

 - เพิ่มเรื่อง "ข่าวผู้ป่วยตะโกนแต่ไม่มีใครได้ยิน" ในส่วนความเป็นมา
 - ระบุชัดเจนเรื่อง "การกระพริบตา" และ "การกำแบมือ" ในส่วน Scope
 - เน้นเรื่อง "Alternative Communication" (การสื่อสารทางเลือก) ในส่วนนโยบายครับ
